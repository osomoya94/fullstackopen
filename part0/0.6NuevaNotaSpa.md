sequenceDiagram
    participant browser
    participant server

    Note right of browser: El usuario escribe la nota y hace clic en Save
    Note right of browser: El código JavaScript intercepta el clic y evita que la página se recargue
    Note right of browser: JavaScript dibuja la nueva nota en la pantalla inmediatamente

    browser->>server: POST https://studies.cs.helsinki.fi/exampleapp/new_note_spa
    activate server
    Note left of server: El servidor guarda la nueva nota en su base de datos
    server-->>browser: HTTP 201 Created (Mensaje de confirmación)
    deactivate server
    
    Note right of browser: El proceso termina aquí. No hay más peticiones GET.
